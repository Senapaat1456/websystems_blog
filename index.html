Blog Entry: 3/9/2020

Adding Sounds to Websites

I was currious on how one could add sound to a webpage. I found two different implementations. The first was to set up an anchor to play sound on a click. This is done with javascript.

The javascript looks like this:

<script type="text/javascript"> function playSound(soundfile){ document.getElementById("Sound_Text").innerHTML="<embed src=\""+soundfile+"\" hidden=\"true\" autostart=\"true\" loop=\"false\" />"; } </script>

and the anchor looks like this:

<a href="#" onclick="playSound('cow_moo.mp3');"> Click here to hear a sound

Click here to hear a sound

The other part I was intrested in was playing a sound when you hover over an element. The implementation of this is similar to above. It uses the exact same javascript function, but it uses the 'onmouseover' attribute instead of an anchor. This can work on every element that shows up on the webpage, but for this example I will use an image.

The code for the image above looks like:

<img src="https://ichef.bbci.co.uk/images/ic/704xn/p0686krr.jpg" onmouseover="playSound('cow_moo_2.mp3')" width=25%>

My source for this info can be found at livewire.com

Embedding Videos

The simplest approach to embedding videos in to simply embed a YouTube video into the web page. Some example code can be see bellow:

<iframe width="560" height="315" src="https://www.youtube.com/embed/dQw4w9WgXcQ?controls=0" frameborder="0" allow="accelerometer; autoplay; encrypted-media; gyroscope; picture-in-picture" allowfullscreen"> </iframe>
